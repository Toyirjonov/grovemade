# GROVEMADE — Статический лендинг (макет)

Коротко:

Это простой статический проект — HTML/CSS-верстка лендинга на основе макета Figma.

- **Макет (Figma)**: https://www.figma.com/design/k8i3wwEPgK0LHP1WGmldF9/GROVEMADE?node-id=0-1&p=f&t=TezjQhgIjEG2sLTV-0

Описание:

Проект содержит минимальную структуру для фронтенд-верстки: `index.html`, стили в `css/style.css` и папку `img` для изображений. Цель — сверстать и протестировать статическую версию дизайна из Figma.

Быстрый просмотр (локально):

- Откройте `index.html` в браузере двойным кликом (подходит для быстрой проверки).
- Для локального HTTP-сервера (чтобы корректно работать с относительными путями и CORS), используйте Python (если установлен):

```powershell
cd .\
python -m http.server 8000
# Затем откройте http://localhost:8000 в браузере
```

Структура проекта:

- `index.html` : главная страница.
- `css/style.css` : основные стили проекта.
- `img/` : изображения, используемые на странице.
- `README.md` : этот файл с инструкциями.
